Bienvenido a OpenClaw

OpenClaw es tu propio asistente de IA autohospedado. Convierte tu computadora en un servidor inteligente capaz de conectarse a Telegram, navegar por la web y ejecutar tareas complejas de forma autónoma.

Guía para Principiantes

No necesitas ser programador. Esta guía te llevará paso a paso desde "cero" hasta tener tu propio bot de IA funcionando en Telegram.

Conceptos Clave

Terminal

La ventana negra donde escribes órdenes. En Windows es "PowerShell" o "CMD", en Mac es "Terminal". Es el volante de tu computadora.

Gateway

El "cerebro" de OpenClaw. Es el programa que corre en segundo plano esperando tus mensajes.

JSON

Archivos de configuración. Son simples archivos de texto donde guardas tus claves secretas y preferencias.

Node.js

La tecnología base. Necesitas tenerlo instalado para que OpenClaw pueda funcionar.

1 Instalación

Antes de empezar en Windows

  • Versión: Windows 10 (versión 2004 o superior) o Windows 11.
  • Permisos: Debes poder abrir PowerShell como Administrador.
  • WSL2 (Opcional pero recomendado): OpenClaw funciona mejor si activas el Subsistema de Windows para Linux.
  • Node.js: El instalador intentará instalarlo si no lo tienes, pero es mejor tenerlo listo.

Selecciona tu sistema operativo y copia el comando correspondiente en tu terminal. El instalador automático se encargará de todo.

Mac / Linux Bash
bash — 80x24 
curl -fsSL https://openclaw.ai/install.sh | bash
PowerShell
PowerShell 
iwr -useb https://openclaw.ai/install.ps1 | iex
CMD (Símbolo sistema)
cmd.exe 
curl -fsSL https://openclaw.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

Verificación

Si la instalación fue exitosa, este comando mostrará "running".

openclaw gateway status

2 Configuración Inicial

Ejecuta el asistente de configuración para conectar tu proveedor de IA (como OpenAI o Anthropic).

Terminal 
openclaw onboard

Pasos del Asistente

  • Selecciona QuickStart.
  • Elige tu proveedor de IA (ej. OpenAI).
  • Pega tu API Key (la clave que empieza con 'sk-...' para OpenAI).

Panel de Control (Dashboard)

Para ver la interfaz gráfica en tu navegador:

openclaw dashboard

3 Conectar Telegram

Paso A: BotFather

  1. Abre Telegram y busca @BotFather.
  2. Envía /newbot.
  3. Sigue los pasos para darle nombre a tu bot.
  4. Importante: Copia el API TOKEN que te da al final.

Paso B: Configuración

Abre el archivo .openclaw/openclaw.json en tu carpeta de usuario y edita la sección de Telegram: 

"telegram": {
  "enabled": true,
  "botToken": "PEGA_AQUI_TU_TOKEN",
  "dmPolicy": "pairing"
}

4 Seguridad (Pairing)

Por seguridad, tu bot ignorará a cualquiera (incluso a ti) hasta que lo autorices. Este proceso se llama "Pairing".

  1. 1

    Saluda a tu bot

    Envía un "Hola" a tu bot en Telegram. Él te responderá con un código (ej. XY123AB).

  2. 2

    Autoriza en Terminal

    Ejecuta el comando de aprobación con tu código:

    openclaw pairing approve telegram [CODIGO]

5 Herramientas

Browser (Navegador)

Permite a la IA navegar por internet, leer páginas web y extraer información en tiempo real.

Exec (Comandos)

Permite a la IA ejecutar comandos en tu computadora, mover archivos y correr scripts.

! Solución de Problemas

Error: "command not found"

Cierra y vuelve a abrir la terminal. A veces el sistema necesita reiniciarse para encontrar el nuevo programa instalado.

El bot no responde

Ejecuta openclaw doctor para que el sistema busque errores automáticamente.
