¿Usuario de Windows? Probá esto primero

Muchos problemas de conexión y lentitud se resuelven refrescando la configuración del sistema.

Cómo ejecutar los comandos:
1. Presiona la tecla Windows y escribe "CMD".
2. En los resultados, haz clic con el botón derecho del mouse sobre "Símbolo del sistema" y selecciona "Ejecutar como administrador".
3. Copia y pega los siguientes comandos uno por uno.

Administrador: Símbolo del sistema
_
□
✕

REM 1. Solucionar problemas de conexión a internet (DNS)

C:\WINDOWS\system32> ipconfig /flushdns

REM 2. Renovar la dirección IP (Conexión inestable)

C:\WINDOWS\system32> ipconfig /release && ipconfig /renew

REM 3. Reparar archivos corruptos de Windows (Lentitud/Errores)

C:\WINDOWS\system32> sfc /scannow

REM 4. Resetear configuración de red completa (Si nada funciona)

C:\WINDOWS\system32> netsh winsock reset

REM 5. Reiniciar la PC inmediatamente (Borra fallos temporales)

C:\WINDOWS\system32> shutdown /r /t 0

¿Usuario de Linux? Probá esto primero

Antes de iniciar una sesión remota, ejecuta estos comandos en tu terminal. A menudo resuelven los problemas más comunes de conectividad y rendimiento.

user@linux:~

# 1. Actualizar repositorios y sistema (Corrige fallos de dependencias)

$ sudo apt update && sudo apt upgrade -y

# 2. Liberar espacio en disco (Elimina paquetes obsoletos)

$ sudo apt autoremove && sudo apt clean

# 3. Diagnóstico de red (Verificar si hay salida a internet)

$ ping -c 4 8.8.8.8

# 4. Reiniciar servicios de red (Si no tienes internet)

$ sudo systemctl restart NetworkManager

# 5. Ver logs de errores críticos recientes

$ journalctl -p 3 -xb
