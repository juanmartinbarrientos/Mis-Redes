Tech Cheat Sheet

Comandos rápidos, bypass de restricciones y configuraciones esenciales para resolver problemas técnicos en tiempo récord.

Instalación de Windows 11

Bypass TPM & Secure Boot

Permite instalar Win11 en hardware no compatible (sin chip TPM 2.0). Ejecutar en la primera pantalla de instalación.

  1. Presiona Shift + F10 para abrir CMD.
  2. Escribe regedit y presiona Enter.
  3. Navega a: HKEY_LOCAL_MACHINE\SYSTEM\Setup
  4. Crea una nueva clave llamada LabConfig
  5. Dentro, crea tres valores DWORD (32-bit):
BypassTPMCheck = 1
BypassRAMCheck = 1
BypassSecureBootCheck = 1

Bypass "Cuenta de Microsoft" (OOBE)

Omite la exigencia de conectarse a internet y usar una cuenta Microsoft al configurar el equipo nuevo (Out Of Box Experience).

  1. Llega a la pantalla de "Vamos a conectarte a una red".
  2. Presiona Shift + F10 para abrir CMD.
  3. Escribe el siguiente comando y presiona Enter:
OOBE\BYPASSNRO

El equipo se reiniciará. Vuelve a hacer los pasos y ahora aparecerá el botón "No tengo internet" -> "Continuar con la configuración limitada".

Recuperación y Accesos

Restablecer Contraseña Local (Truco Utilman)

Si un usuario olvidó la contraseña local de Windows. Requiere un USB booteable de instalación de Windows.

Fase 1: Preparación (Boot USB)

  1. Bootea con el USB de instalación de Windows.
  2. En la primer pantalla, presiona Shift + F10.
  3. Identifica la letra del disco (usualmente C: o D:). Escribe los comandos:
c:
cd windows\system32
ren utilman.exe utilman.exe.bak
copy cmd.exe utilman.exe

Cierra todo y reinicia la PC normalmente.

Fase 2: Ejecución de NET USER

  1. En la pantalla de bloqueo, haz clic en el icono de Accesibilidad (esquina inferior derecha).
  2. ¡Se abrirá el CMD con privilegios SYSTEM!
  3. Escribe el comando para cambiar la clave:
net user [nombre_usuario] [nueva_contraseña]

Ejemplo: net user Administrador 1234. Luego inicia sesión.
No olvides revertir el cambio de utilman.exe luego.

Rendimiento y Mantenimiento

Sensor de Almacenamiento

Automatiza la limpieza de temporales y papelera sin instalar software de terceros (como CCleaner).

  • Ir a Configuración > Sistema > Almacenamiento.
  • Activar "Sensor de almacenamiento".
  • Clic en configurar:
    • - Ejecutar: Cada semana.
    • - Papelera: Borrar si tienen > 14 días.
    • - Descargas: Borrar si no se abren en 30 días (opcional).

Reparación de Imagen (CMD Admin)

Si Windows falla, hay pantallazos azules o errores de sistema, corre estos comandos en orden.

1. Chequea y repara archivos de sistema
sfc /scannow
2. Repara la imagen de Windows usando WinUpdate
DISM /Online /Cleanup-Image /RestoreHealth

Apagar "Fast Startup"

Soluciona problemas de discos que no se montan bien en dual-boot o cuelgues al iniciar.

Panel de Control > Opciones de energía > Elegir comportamiento de botones > Cambiar configuración no disponible > Desmarcar "Activar inicio rápido".

Resetear Red

Soluciona el típico "Conectado, sin internet".

ipconfig /flushdns
netsh winsock reset

DNS & Adaptadores

El atajo definitivo para abrir "Conexiones de red" sin navegar por 50 menús del nuevo Windows.

Win + R ➜
ncpa.cpl

Click derecho en el adaptador > Propiedades > IPv4.

DNS Recomendados:

"Modo Dios" (God Mode)

Crea una carpeta que unifica TODAS las configuraciones del Panel de Control de Windows en un solo lugar. Ideal para llevar en un pendrive o dejar oculta en tu usuario admin.

  1. Crea una nueva carpeta en el Escritorio.
  2. Renómbrala pegando exactamente el siguiente código:
GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}

Actualizar Todo (Winget)

La navaja suiza moderna. Actualiza todos los programas de terceros del equipo de un solo golpe. Ideal para solucionar bugs y lentitud por software obsoleto.

winget upgrade --all

Limpieza WinSxS (GBs extra)

¿Disco C: lleno? Esto purga las copias de seguridad de Windows Update incrustadas profundamente en el sistema. Recupera GBs que el limpiador normal ignora.

DISM /Online /Cleanup-Image
/StartComponentCleanup /ResetBase

Reparar UI (PowerShell)

El rescate absoluto cuando el Menú Inicio no abre, la barra se congela o las apps nativas crashean sin motivo (Requiere abrir PowerShell como Admin).

Get-AppxPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

Atajos de Teclado Pro

Gestión de Ventanas

Win + Z

Olvídate de arrastrar ventanas. Abre los Snap Layouts y permite asignar ventanas a zonas específicas usando números. Vital para manejar múltiples sesiones RDP o logs en monitores ultrawide.

Copiar Ruta de Acceso

Ctrl + Shift + C

En el Explorador de archivos, copia la ruta completa entre comillas al portapapeles.

💡 Uso: Pegar directo en scripts de PowerShell o configs JSON sin añadir comillas manualmente.

Salvavidas Gráfico

Win + Ctrl + Shift + B

¿Pantalla negra o driver congelado en sesión remota? Reinicia el controlador gráfico sin cerrar tus aplicaciones. Escucharás un "beep" y la pantalla parpadeará. Magia pura.

Clipboard Histórico

Win + V

Activa el historial del portapapeles. No solo guarda lo último copiado; puedes anclar (pin) elementos.

💡 Uso: Deja ancladas esas IPs de servidores o comandos CLI que usas todo el día.

Multitarea Terminal

Alt + Shift + D

Dentro de Windows Terminal, divide el panel actual. Puedes tener tu servidor corriendo a la izquierda y un tail -f del log a la derecha en la misma ventana.

BONUS

CMD Admin Ninja

El combo más rápido para abrir Terminal como Administrador. Olvídate del ratón.

Win + X suelta, y presiona A

En menos de 1 segundo tienes el prompt del UAC. Una vez que tu memoria muscular lo aprende, no hay vuelta atrás. ⏳

Comandos Linux Pro

ncdu

Olvida el du -h. Analiza qué carpeta se comió tu disco duro de forma visual e interactiva.

ncdu

tldr

¿Te da pereza leer el man? Muestra solo los ejemplos prácticos que realmente usas.

tldr [comando]

lsof

"File system busy"? Encuentra qué proceso tiene secuestrado tu archivo. Ideal para debug.

lsof [archivo]

pgrep / pkill

Busca o mata procesos por nombre directamente. Adiós a buscar el PID manualmente con ps aux.

pkill [nombre]

rsync

El rey de las copias. Sincroniza archivos local o remotamente con precisión quirúrgica.

rsync -avz [orig] [dest]

IP Pública

Tu IP pública al instante, directo en terminal, sin abrir navegadores pesados ni webs de terceros.

curl ifconfig.me

tree

Visualiza la estructura de tus directorios como un árbol. Perfecto para documentar proyectos.

tree -L 2

dstat

Estadísticas en tiempo real de CPU, RAM, disco y red. Todo consolidado en una sola pantalla visualmente amigable.

dstat
🔥 BONUS TRACK

Corrector 'fc'

¿Te equivocaste en un comando larguísimo? Escribe fc, edítalo en tu editor favorito (nano/vim) y al guardar se ejecuta corregido automáticamente. Magia pura.

fc

autojump (j)

Aprende tus rutas más usadas. En vez de escribir largas rutas como cd /var/www/html/proyecto_largo, simplemente usas el comando j proyecto y ¡bum! Estás ahí al instante.

j proyecto
